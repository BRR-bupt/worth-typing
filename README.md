# worthTyping
让打字更有意义！

这是一个打字网站，文章可选择古文、现代文、英文，所选篇目具有一定的文学价值，并附有古文注释与知识解析，让用户在打字测速的同时，获得有意义的文化熏陶。

## Demo
[网页链接](https://worthtyping.netlify.app/)

## 🚀 Features

- 🎪 基于 **vue3** 框架开发
- ⚡ 基于 **vite** 开发，获取更快速的启动速度
- 🦾 完全使用 **typescript** 提供完备的代码类型检查
- 🔋 使用**pinia** 提供状态管理服务，极大地简化了组件间通信

## 📦 Install

```bash
pnpm install
```

## 🦄  serve
